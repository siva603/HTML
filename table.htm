HTML Table Tag

    
    HTML tables allow web developers to arrange data into rows and columns. We must write everything inside the table tag.

    <table> : Defines an HTML table, which is used to organize data into rows and columns.

    <tbody> : Groups the main content (data rows) of an HTML table. 

    <thead> : Groups the header content (table heading ) of an HTML table. 

    <th> : Defines a header cell (table heading) within a table. 

    <td> : Represents a data cell within an HTML table. 

    <tr> : Defines a row within an HTML table.
